
```bash

```

3. Set up environment variables:
Create a `.env` file in the root directory and add the following:
```env
DATABASE_URL=your_mongodb_url
JWT_SECRET=your_jwt_secret
```

4. Run the development server:
```bash
npm run dev
# or
yarn dev
```
